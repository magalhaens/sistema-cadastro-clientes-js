# Sistema de cadastro de clientes

Sistema simples de cadastro de clientes desenvolvido com HTML, CSS e JavaScript puro.

## Sobre o projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais de JavaScript, principalmente funções, arrays, eventos e manipulação do DOM.

A aplicação permite cadastrar nomes de clientes, armazená-los em um array e exibi-los dinamicamente na tela.

## Funcionalidades

- Cadastrar clientes pelo input
- Validar campo vazio
- Salvar clientes em um array
- Mostrar clientes cadastrados na tela
- Limpar o input após o cadastro
- Manter o foco no input depois de cadastrar

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Conceitos praticados

- Funções
- Arrays
- Eventos de clique
- `addEventListener`
- `querySelector`
- `createElement`
- `appendChild`
- `textContent`
- `if`
- `trim()`
